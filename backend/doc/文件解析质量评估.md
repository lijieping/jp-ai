| 类别       | 指标             | 合格阈值（经验值） | 计算方式                                   | 阶段     
| -------- | -------------- | --------- |----------------------------------------|--------|
| **完整性**  | 文本提取率          | ≥ 95 %    | OCR 字数 / 人工标注总字数                       | 解析 √   |
| **准确性**  | OCR 字符错误率（CER） | ≤ 3 %     | 误识字符 / 总字符                             | 解析  √    |
| **可读性**  | 乱码/特殊符号占比      | ≤ 0.5 %   | 匹配“�”“”等符号出现次数 / 总字符             | 清洗     |
| **重复度**  | 段落 SimHash 重复率 | ≤ 10 %    | 重复段落数 / 总段落数                           |        |
| **结构保真** | 表格结构还原率        | ≥ 90 %    | 成功解析成 HTML/CSV 的表格数 / 视觉可见表格数    | 解析     |
| **语义密度** | 停用词外有效词占比      | ≥ 60 %    | (总词数 - 停用词数) / 总词数                | 混合索引构建 |
| **元数据**  | NER 实体覆盖率      | ≥ 80 %    | 自动抽取实体数 / 人工标注实体数                   |
